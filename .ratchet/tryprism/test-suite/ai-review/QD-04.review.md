## Task

You are reviewing the quality of the PDF report generated by the TryPrism application. Generate a PDF for a Type 4 result (wing 5, tritype 459) using both the English and Chinese language settings, then evaluate each according to the rubric below.

---

## Rubric

| Score | Description |
|-------|-------------|
| **5** | Professional report quality — clear sections, good typography, chart/visualization included, all result sections present, looks like a paid assessment report |
| **4** | Clean and complete — all required sections present, reasonable layout, might lack visual polish but clearly professional |
| **3** | Readable but plain — text-only, minimal formatting, functional but not impressive; would pass as an internal document |
| **2** | Missing important sections or formatting issues make it hard to navigate |
| **1** | Broken formatting, missing sections, unreadable, or PDF generation failed |

**Pass threshold: 4 or above.**

## Evaluation Criteria

### 1. Layout and Visual Hierarchy (weight: 25%)

Evaluate the PDF's structure and visual presentation:

- [ ] **Cover / header section** — Does the PDF have a clear title/header that identifies it as a TryPrism Enneagram report? Does it include the user's primary type prominently?
- [ ] **Section separation** — Are the major sections clearly separated? (e.g., dividers, spacing, page breaks)
- [ ] **Typography** — Is the font legible? Are headings visually distinct from body text? Is font size appropriate (minimum 10pt for body, 14+ for headings)?
- [ ] **Color usage** — Does the PDF use the type's accent color or the prismatic color scheme? Or is it plain black and white?
- [ ] **Page margins** — Are margins reasonable (not too cramped, not excessive)?
- [ ] **Page numbering** — If more than one page, are pages numbered?

### 2. Content Completeness (weight: 35%)

Verify that the following sections are present and populated:

- [ ] **Primary type** — Type number (e.g., "Type 4") and name (e.g., "The Individualist") are shown
- [ ] **Wing** — Wing notation shown (e.g., "4w5")
- [ ] **Type description** — Full description text (200+ words)
- [ ] **All 9 type scores** — Score for each type displayed (as a chart, table, or list)
- [ ] **Growth direction** — The integration/growth arrow is mentioned or shown (e.g., "Type 4 grows toward Type 1")
- [ ] **Stress direction** — The disintegration/stress arrow is mentioned or shown (e.g., "Under stress, Type 4 moves toward Type 2 patterns")
- [ ] **Tritype** — Tritype is noted (e.g., "459")
- [ ] **Strengths** — At least 3 key strengths listed
- [ ] **Growth suggestions** — At least 2–3 actionable suggestions included
- [ ] **Test metadata** — Test date and/or mode (quick/full) included

### 3. Score Visualization (weight: 20%)

- [ ] **Is a chart or visual score representation included in the PDF?**
  - A bar chart, radar chart, or even a styled table showing all 9 scores is required
  - Plain text listing scores (e.g., "Type 4: 28%") without any visual representation scores lower
- [ ] **Is the chart readable at PDF resolution?** (not blurry/rasterized at low resolution)
- [ ] **Does the chart use the type's accent colors or the prismatic palette?**
- [ ] **Can the reader immediately see which type scored highest?**

### 4. Bilingual Support (weight: 10%)

- [ ] **English PDF is fully in English** — no Chinese characters appear in the English PDF
- [ ] **Chinese PDF is fully in Chinese** — no untranslated English strings appear in the Chinese PDF
- [ ] **Chinese characters render correctly** — no tofu (□□□) or garbled characters in the Chinese PDF

### 5. Shareability and Professionalism (weight: 10%)

Would you share this PDF with a friend, therapist, or employer without embarrassment?

- [ ] The PDF does not look like a raw data dump or debug output
- [ ] There are no error messages, undefined values, or [object Object] strings visible
- [ ] The filename is meaningful (e.g., `tryprism-type4-results.pdf`, not `download.pdf`)
- [ ] The PDF could plausibly come from a paid personality assessment service
- [ ] Branding / app name is visible somewhere in the PDF
